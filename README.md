# To Do List
**A Command Line To Do List App**

## Setup/Installation


## Running Tests
To run tests, from the root directory use the command:  
```$ rake spec```

## Running the Program
To start running the program, from the root directory:  
```$ ruby lib/main.rb```

## Dependencies
ruby 2.6.2  
RSpec 3.8
